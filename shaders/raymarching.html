RAYMARCHING

📏 The Raymarch Loop

Sphere tracing: from the camera, repeatedly step along the ray by exactly the distance to the nearest surface (the SDF value) — you can never overshoot, so you converge onto the surface. Too few steps and grazing rays fall short.

The whole renderer

for i in 0..STEPS:
  d = map(ro + rd * t);   // distance to nearest surface
  if (d < ε) break;   // hit
  t += d;   // safe to jump exactly d

STEPS is a constant loop bound → the buttons rebuild the shader.

📝 Click to see: the raymarch loop 
float map(vec3 p) { return length(p) - 1.0; }   // a sphere
for (int i = 0; i < ${steps}; i++) {           // bound MUST be constant
  vec3 p = ro + rd * t;
  d = map(p);
  if (d < 0.002 || t > 12.0) break;
  t += d;
}
// camera ro/rd are built by the shared RM_CAM kit; mouse = orbit.
Step along the ray by the SDF value until you hit. Drag to orbit.

🧊 SDF Primitives & Smooth-Min

Sphere, box and torus each have a one-line distance function. Combine them with min (union), max/- (subtract), or the polynomial smooth-min that melts them into organic blends.

📝 Click to see: SDF primitives & smin 
float smin(float a, float b, float k){
  float h = clamp(0.5 + 0.5*(b-a)/k, 0.0, 1.0);
  return mix(b, a, h) - k*h*(1.0-h);
}
// union: min(a,b)  subtract: max(-a,b)  blend: smin(a,b,k)
// normals via central differences of map() (e.xyy swizzle trick).
One-line primitives, combined with min / max / smin.

💡 Normals, Diffuse & Soft Shadows

The surface normal is the gradient of the SDF (central differences). Lambert N·L gives diffuse; marching a ray toward the light gives shadows — and tracking the closest miss turns hard shadows into soft penumbrae for free.

📝 Click to see: lighting & soft shadows 
// soft shadow: along the shadow ray, keep the smallest h/t
sh = min(sh, 10.0 * h / st);   // h = SDF, st = distance travelled
// hard shadow is the same loop without that running-minimum.
Normal = ∇map; Lambert + a shadow march. The light moves.

🌑 Ambient Occlusion & Fog

Five short marches along the normal estimate how "boxed in" a point is — that ambient occlusion darkens creases and contact points and is what makes the render read as solid. Exponential fog adds depth.

📝 Click to see: AO & fog 
for (int i = 0; i < 5; i++) {
  float h = 0.02 + 0.12 * float(i);
  o += (h - map(p + n * h)) * s;   // expected vs actual distance
  s *= 0.7;
}
occ = clamp(1.0 - 1.6 * o, 0.0, 1.0);
// fog: mix(col, sky, 1.0 - exp(-k * t * t)).
Short normal marches → contact shadows; exp fog → depth.

🌀 Mandelbrot & Julia

Escape-time fractals: iterate z = z² + c and colour by how fast it escapes. For the Mandelbrot set c is the pixel; for a Julia set c is fixed (drive it with the mouse). Uses precision highp float for a clean deep zoom.

📝 Click to see: the escape-time fractal 
for (int i = 0; i < ${iter}; i++) {            // iteration count = constant
  z = vec2(z.x*z.x - z.y*z.y, 2.0*z.x*z.y) + cc;
  if (dot(z, z) > 64.0) break;
  it += 1.0;
}
float sm = it + 1.0 - log2(0.5 * log2(dot(z,z)));  // smooth colour
// Julia: z starts at the pixel, cc is the (mouse-driven) constant.
z = z² + c, coloured by escape speed. Slow auto-zoom.

🪐 Mandelbulb (3D fractal)

The Mandelbrot's 3D cousin: a distance-estimator raymarch of the power-8 spherical iteration. The running minimum radius is an orbit trap used to colour it. precision highp float.

📝 Click to see: the Mandelbulb DE 
dr = pow(r, P-1.0) * P * dr + 1.0;          // distance-estimator
float zr = pow(r, P);
theta *= P; phi *= P;
z = zr * vec3(sin(theta)*cos(phi), sin(theta)*sin(phi), cos(theta)) + pos;
return 0.5 * log(r) * r / dr;                // the DE
// P (power) and DE iterations are constants → rebuild on change.
A distance-estimated 3D fractal. Orbit with the mouse.

♾️ Repeated Worlds (Domain Repetition)

mod(p, s) on the position evaluates one small SDF as if it were tiled infinitely — an endless lattice for zero extra cost. abs(mod(...)) mirrors the cells into a kaleidoscopic tunnel you fly through.

📝 Click to see: domain repetition 
float map(vec3 p) {
  p.z += u_time * 1.5;            // fly forward
  vec3 q = mod(p, 2.0) - 1.0;     // ← infinite tiling
  return min(length(q) - 0.45, sdBox(q, vec3(0.18)));
}
// mirror variant: vec3 q = abs(mod(p, 4.0) - 2.0) - 1.0;
mod() on position → an infinite world from one object.

🏔️ Mini-Project: Raymarched Scene

Everything composed: smin-blended primitives on a ground plane, a moving light with soft shadows, ambient occlusion, exponential fog, a cosine-palette material and a gamma pass — a game-ready vignette / skybox / loading backdrop.

📝 Click to see: the composed scene 
col = base * (0.12 + 0.95 * diff * softShadow) * ao
    + pow(diff, 32.0) * softShadow * 0.6;
col = mix(col, sky, 1.0 - exp(-0.018 * t * t));      // fog
col = pow(col, vec3(0.4545));                         // gamma
// presets swap the cosine palette (crystal / lava / ice).
smin + soft shadow + AO + fog + palette + gamma — one shader.
