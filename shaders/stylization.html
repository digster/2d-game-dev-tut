🖍️ Toon / Cel Shading

The cartoon look in one move: take the smooth lighting and quantise the value channel into a few hard bands, keeping the hue. Add a luminance-gradient outline and you have the anime/cel-shaded staple.

Band the value, keep the hue

L = luminance(colour);
q = floor(L * bands) / (bands - 1.0);   // hard steps
out = (colour / L) * q;   // re-apply banded value to the hue

Band count is a shader constant → the buttons rebuild via setFrag, the same structural-vs-continuous rule from the noise tiers.

📝 Click to see: the toon shader 
function buildFrag(bands, rim) {
  return NPR_HEAD + NPR_SCENE + `void main() {
  vec3 c = scene(gl_FragCoord.xy / u_resolution);
  float L = lum(c);
  float B = ${bands}.0;
  float q = floor(L * B) / (B - 1.0);          // hard value bands
  vec3 outc = clamp((c / max(L, 1e-3)) * q, 0.0, 1.0);
  // ... luminance-gradient edge ink when rim is on ...
  gl_FragColor = vec4(outc, 1.0);
}`;
}
function buildFrag(bands: number, rim: number): string {
  return NPR_HEAD + NPR_SCENE + `/* band L into 'bands' steps, keep hue, ink edges */`;
}
Quantise the value into hard bands, keep the hue, ink the silhouette.

✏️ Sobel / Roberts Edge Ink

Edge detection turns any image into line art. Sobel is a 3×3 weighted gradient; Roberts is a cheap 2×2 diagonal one. Threshold the gradient magnitude and you get ink — on paper, or over the colour scene.

The gradient magnitude is the edge

g = length(vec2(∂L/∂x, ∂L/∂y)). Big where luminance changes fast (silhouettes, shading boundaries), ~0 on flat regions — smoothstep it into a clean ink mask.

📝 Click to see: the edge-detect shader 
// Sobel: 9 luminance taps → gx, gy → length() → ink mask.
const sobel = `... gx = (L20+2L21+L22) - (L00+2L01+L02); ...
  float g = length(vec2(gx, gy));`;
const ink = `smoothstep(0.10, 0.22, g)`;
// Roberts is the cheap 2x2 alternative: length(vec2(a-d, b-c)).
Threshold the luminance-gradient magnitude → clean line art.

✒️ Cross-Hatch (scene-wide)

The engraving / pencil look: as a pixel gets darker, unlock another layer of rotated parallel lines. Computed in screen space so the strokes stay a constant physical size — exactly how a real etching behaves.

📝 Click to see: the cross-hatch shader 
float k = 1.0;
if (L < 0.85) k = min(k, step(0.5, fract((g.x + g.y) * F)));
if (L < 0.62) k = min(k, step(0.5, fract((g.x - g.y) * F)));
if (L < 0.40) k = min(k, step(0.5, fract( g.x * F * 1.5)));
// darker → more line layers stack → reads as deeper shadow
// u_param scales the nib frequency F (continuous → uniform, not rebuild).
Darker luminance stacks more rotated line layers — engraving.

⚫ Halftone / Ben-Day Dots

Print reproduces tone with dots of varying size. One rotated screen gives a mono halftone; rotating a separate screen per channel gives the CMYK comic-book / Lichtenstein Ben-Day look.

📝 Click to see: the halftone shader 
float dotScreen(vec2 g, float ang, float tone){
  vec2 cell = fract(rot2(g, ang) * F) - 0.5;
  float r = sqrt(tone) * 0.62;          // darker → bigger dot
  return smoothstep(r - 0.05, r + 0.05, length(cell));
}
// CMYK: dotScreen(.r,0.26), dotScreen(.g,1.30), dotScreen(.b,2.30).
Tone becomes dot size; rotate a screen per channel for Ben-Day.

🖌️ Kuwahara Oil Paint

An edge-preserving filter that reads as paint. For each pixel, look at four overlapping quadrants and output the mean of the lowest-variance one — flat regions smooth, edges stay crisp, like a palette knife.

Why it preserves edges

Low variance = "this side is a calm region". A pixel straddling an edge copies the calm side instead of averaging across the boundary, so the boundary survives. The window K is a loop bound → structural → setFrag.

📝 Click to see: the Kuwahara shader 
const int K = ${k};
void region(vec2 uv, vec2 px, vec2 d, out vec3 mean, out float varr){
  for (int j=0;j<=K;j++) for (int i=0;i<=K;i++) { /* mean + variance */ }
}
// output = mean of the quadrant with the smallest variance
// 4 quadrants sampled from the procedural scene() — no texture needed.
Keep the flattest of four quadrants → painterly, edges intact.

🔡 ASCII / Dither Art

Two structure-changing stylizers. ASCII collapses each grid cell to one luminance value and draws a brightness-keyed glyph (terminal art). Dither applies an interleaved-gradient 1-bit threshold — the eye fuses the noise into shades.

📝 Click to see: the ASCII / dither shader 
vec2 g = uv * vec2(${cells}.0);
vec3 cc = scene((floor(g) + 0.5) / vec2(${cells}.0));   // one sample / cell
float lvl = floor(lum(cc) * 5.0);                        // glyph weight
float bars = step(0.5, fract(fract(g).y * lvl + 0.25));  // strokes
// dither: step(interleavedGradientNoise(fragCoord), luminance) → 1-bit.
Collapse cells to glyphs, or 1-bit dither — structure, not just tone.

🎛️ Mini-Project: Stylization Stack

The capstone: one scene, a switchable NPR pipeline. Flip between the cartoon, pencil, comic-print, painterly and terminal looks live — exactly the kind of art-direction toggle a game's render settings expose.

📝 Click to see: the stylization stack 
const STYLE = { toonink: `...`, hatch: `...`, halftone: `...`,
                oil: `...`, ascii: `...` };
function buildFrag(style) {
  return NPR_HEAD + NPR_SCENE + `void main(){ ${STYLE[style]} }`;
}  // switching the style rebuilds the shader (structural)
type Style = 'toonink' | 'hatch' | 'halftone' | 'oil' | 'ascii';
One scene, five non-photorealistic pipelines — pick the art direction.
