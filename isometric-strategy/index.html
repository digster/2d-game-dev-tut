🎯 What You'll Build

This track teaches you how to build a real-time strategy game from scratch — the kind of game where you scout a map, harvest resources, build a base, train units, and outmaneuver an opponent. Think Age of Empires II, Red Alert 2, or StarCraft.

🎬 The final game (by the end of the Advanced tier)

  • An isometric tile map you can pan and zoom around
  • Workers that harvest from resource tiles and return to drop-off buildings
  • Buildings you can place on valid terrain, with multi-tile footprints
  • Combat units with HP, attack range, and pathfinding around obstacles
  • A drag-box selection + right-click commands for crisp RTS feel
  • Fog of war that hides unexplored territory
  • A scripted AI opponent that scouts, expands, and attacks
💡 Why this is the first end-to-end track: An RTS touches almost every concept in the Fundamentals guide — vectors (unit movement), matrices (isometric projection), pathfinding (A*), tilemaps, AI state machines, and performance optimization. Building one is the best way to see how the fundamentals compose into a real game.

🗺️ Track Roadmap

The track is split across five tiers, each in its own file so it stays focused and quick to load. Each tier ends with a playable mini-project that becomes the foundation of the next tier.

📚 Recommended Prerequisites

You can start this track cold, but you'll move faster if you've already worked through these topics from the Fundamentals guide:

Strongly recommended before BEGINNER

  • Vectors: Position & Direction — we use 2D vectors everywhere
  • Matrix Transformations — iso projection is a 2D linear transform
  • Trigonometry for Game Developers — for unit facing

Strongly recommended before INTERMEDIATE

  • Linear Interpolation (Lerp) — smooths tile-to-tile movement
  • Tilemap Systems — we build on this with isometric rendering

Strongly recommended before EXPERT

  • A* Pathfinding — we apply it to the iso grid
  • State Machines — for unit behavior

Strongly recommended before ADVANCED

  • Flow Fields (Crowd Movement) — for group movement
  • Shadow Casting & Vision — for fog of war
  • Spatial Hashing — for 1000+ units

💡 How to Use This Track

👶 If you're new to game dev

Don't start here — start with the Beginner Fundamentals first, then come back. This track assumes you've seen vectors, basic trig, and a canvas before.

🛠️ If you've made small games before

Start at the Beginner tier of this track. The first tier is short and locks in the iso math you'll lean on for the rest of the journey.

🎮 If you've shipped an RTS before

Jump to the Expert or Advanced tier and use the earlier ones as a refresher for the iso math and depth-sort details. The Simulations tier is where the interesting visualizations live.
