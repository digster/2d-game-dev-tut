🎯 What You'll Build

This track teaches 2D voxel worlds — games where the playable area is a grid of cells, each storing a material, and the player can change that grid. There are two famous flavors and we cover each as its own sub-track.

🌲 Terraria sub-track

  • Scrollable side-view world built from a tile grid
  • Mine and place tiles with a mouse cursor
  • Foreground & background layers (Terraria's wall system)
  • Falling sand & gravel (simple cellular rules)
  • Cellular-automaton liquids that flow downhill
  • Tile-based lighting propagated by flood fill
  • Biome-zoned worldgen from Perlin noise

🔥 Noita sub-track

  • Every pixel is a material — no tile-grid approximation
  • Sand piles and slides into gaps
  • Water flows sideways and pools
  • Oil floats on water (density rules)
  • Fire ignites wood and oil, burns down, produces smoke
  • Gases rise, react with materials they touch
  • Material reactions: water + lava → steam + stone
💡 Why two sub-tracks? Terraria voxels are block-discrete: one tile = one material, edited explicitly by the player. Noita voxels are cellular-continuous: every pixel runs a simulation step every frame, all on its own. Different data structures, different update order, different performance constraints. We cover Terraria first because most of its foundations carry over; the Noita sub-track only teaches what's new.

📚 Recommended Prerequisites

You can start this track cold, but you'll move faster if you've already worked through these topics from the Fundamentals guide:

Strongly recommended before the TERRARIA Beginner tier

  • Vectors: Position & Direction — tile coords are 2D ints, camera is a vector
  • Building Your First Game System — the per-frame render loop

Strongly recommended before TERRARIA INTERMEDIATE / EXPERT

  • Tilemap Systems — the canonical tile-grid model
  • Perlin Noise & Terrain Generation — for biome surfaces and caves
  • Collision Detection — AABB player vs. tile grid

Strongly recommended before TERRARIA ADVANCED

  • A* Pathfinding — enemy AI on the (destructible) grid
  • Shadow Casting & Vision — tile-based lighting
  • Spatial Hashing — chunking large worlds

Strongly recommended before the NOITA sub-track

  • The Terraria Beginner tier — Noita assumes you already know tile coords, cameras and grid storage
  • Fluid Dynamics Simulation — grid-based fluid baseline
  • Destructible Terrain — explicitly covers pixel/voxel destruction approaches
  • Thermal Simulation — heat is the seed for fire propagation

💡 How to Use This Track

👶 If you're new to voxel games

Start with the Terraria Beginner tier. It teaches the data structures and rendering loop that every 2D voxel game (including Noita) is built on.

🛠️ If you've shipped a platformer before

The Terraria Beginner tier will be a quick review of tile coords and viewport culling. Skim it, then dig into the Intermediate / Expert tiers where mining, player physics, and worldgen live.

🔥 If you want pure simulation

The Noita sub-track is where the falling-sand physics live. Skim the Terraria Beginner tier first (grid coords and storage carry over verbatim), then jump straight to Noita.
