## New infra: makeMeshToy
The track's first non-fullscreen-quad runner — builds a 56×56 triangulated

Lives in `distortion-demos.js`, stringified as `sh_makeMeshToy`.

## Key learnings
- **GLSL ES 1.00 has no implicit int→float.** A bare JS-interpolated number
  becomes an int literal; `1 * edge` (int×float) is a hard compile error.
  Discipline: every numeric `${n}` feeding float math is `${n}.0`. One real bug
  (`${rim} * edge` in toon) found & fixed.
- **Lazy-mount makes verification async.** A synchronous click-then-read races
  the IntersectionObserver. Reliable protocol: scroll → let a tool round-trip
  elapse (mount + a RAF) → read `info.style.color` / screenshot. The toon bug
  was missed by the sync probe and only caught by the screenshot.
- Preview console buffer accumulates across reloads — screenshots / post-mount
  `info.style.color` are the authoritative signals, not the raw log.

# 2026-05-18 (pt.2) — Fix two broken simulation demos

## Request
Two issues reported on the simulations page: (1) GPU Cloth (Verlet) renders
nothing (black + faint specks); (2) Reaction-Diffusion (Gray-Scott) presets
CORAL/MITOSIS/WORMS all produce the same moiré output. Plan-mode → approved.

## Root causes & fixes (applied to BOTH simulations-demos.js & bundles-simulations.js)
1. **Cloth pin inverted.** Seed maps texel row 0 → top of screen, row 63 →
   bottom; pin test was `ip.y == sz.y-1` (bottom row). Gravity dragged the
   unsupported sheet off-screen. Fix: pin `ip.y == 0`.
2. **Cloth WIND unpinned (found during verification).** Pin gate was
   `u_param < 0.5`; WIND sets u_param=2.0 so the "pinned flag" detached and
   fell. Fix: `u_param < 0.5 || u_param > 1.5` (hold for PINNED 0 & WIND 2,
   release only DROP 1).
3. **Gray-Scott unstable Laplacian.** Un-normalized kernel (orth 1.0, diag
   0.5, center -6.0) with Da=1.0,dt=1.0 → checkerboard eigen ≈-8, update
   factor ≈-7/step → clamp-saturated moiré that swamped feed/kill, so all
   presets looked identical. Fix: canonical normalized kernel (0.2 / 0.05 /
   -1.0), stable with the existing Da/Db/dt.
4. **RD self-development (beyond plan, needed to meet plan's own success
   bar).** (a) Seed was A=1,B=1 in patch → A collapses, no reaction front;
   the lower-feed regimes died. Fixed to canonical A=0,B=1 patch
   (`vec4(1.0-b, b, 0, 1)`). (b) State grid defaulted to 800×450 → sub-pixel
   features; set `stateW:320, stateH:180` so features upscale and read.
5. **Cloth visibility.** Cloth reused the dense-particle POINT_* shaders
   (2px, `v_spd=length(.zw)` = |prev|, meaningless). Added cloth-local point
   shaders: 3.5px, true Verlet speed `length(.xy-.zw)`, brighter color.

## Verified (browser, console-clean)
- Coral = full organic maze; Mitosis = isolated dividing cells; Worms =
  growing filament — three distinct characteristic patterns, no moiré.
- Cloth: PINNED hangs (clearly visible blue sheet), DROP falls, WIND ripples
  a pinned flag, RESET re-seeds. No GL errors.

  pin-row fix made the demo run far enough to exercise WIND.
