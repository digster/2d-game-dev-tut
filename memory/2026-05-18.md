## New infra: makeMeshToy
The track's first non-fullscreen-quad runner — builds a 56×56 triangulated
grid mesh (`a_position`/`a_uv`), runs a user vertex + fragment shader, same
uniforms as `makeShaderToy`. Returns the `lazyToy` handle shape + `rebuild
({vert,frag})` (already in `LAZY_DEMO_METHODS`) → **zero edits to `shared/`**.
Lives in `distortion-demos.js`, stringified as `sh_makeMeshToy`.

## Key learnings
- **GLSL ES 1.00 has no implicit int→float.** A bare JS-interpolated number
  becomes an int literal; `1 * edge` (int×float) is a hard compile error.
  Discipline: every numeric `${n}` feeding float math is `${n}.0`. One real bug
  (`${rim} * edge` in toon) found & fixed.
- **Lazy-mount makes verification async.** A synchronous click-then-read races
  the IntersectionObserver. Reliable protocol: scroll → let a tool round-trip
  elapse (mount + a RAF) → read `info.style.color` / screenshot. The toon bug
  was missed by the sync probe and only caught by the screenshot.
- Preview console buffer accumulates across reloads — screenshots / post-mount
  `info.style.color` are the authoritative signals, not the raw log.
