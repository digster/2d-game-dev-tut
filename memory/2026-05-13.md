# 2026-05-13 — Isometric Strategy Track (Phase 1 scaffolding)

## What changed

Introduced a new content category alongside the existing tier-based Fundamentals: **Game Tracks** —
end-to-end "build this specific game" tutorials. Seeded the category with an **Isometric Real-Time
Strategy** track (AoE2 / Red Alert 2 style).

Phase 1 of the track delivers scaffolding only. Tier files (beginner / intermediate / expert /
advanced / simulations) will land in subsequent phases per the plan at
`~/.claude/plans/along-with-the-proud-dragonfly.md`.

## Files created

- `isometric-strategy/index.html` — track landing page. Shows the 5-tier roadmap with
  "Coming soon" placeholder cards, a "what you'll build" preview, and a prerequisites section
  linking back to Fundamentals topics.

## Files modified

- `index.html` (root) — added a new **🏰 Game Tracks** section between "Choose Your Learning
  Path" and the Complete Table of Contents. Also appended a Game Tracks block to the TOC with
  nested links to each tier of the new track.
- `README.md` — replaced the 3-line stub with a description of both content categories
  (Fundamentals and Game Tracks) plus a pointer to shared assets.

## Architecture decisions made

- **Subdirectory per track** (not flat naming). `isometric-strategy/` holds all track files;
  shared assets are referenced as `../shared/...` from inside the subdirectory.
- **Track-local index page** distinct from the root `index.html`. Acts as the roadmap +
  prerequisites + final-result preview for the track.
- **Inline styles on the track landing page** for the tier-card grid. Kept on that one page
  rather than promoted to `shared/styles.css` — if a second track reuses them, they should be
  promoted then.
- **No changes to `shared/styles.css` or `shared/utils.js`** in Phase 1. Iso helpers
  (`cartToIso`, `isoToCart`, `drawIsoTile`, `pickTileFromMouse`) are deferred to Phase 2 so
  they're added alongside the beginner-tier content that uses them.

## Known follow-ups (per plan)

- **Phase 2 — Beginner tier**: `isometric-strategy/beginner.html` + `beginner-demos.js`, plus
  iso helpers in `shared/utils.js`.
- **Phase 3 — Intermediate tier**, **Phase 4 — Expert**, **Phase 5 — Advanced**, **Phase 6 —
  Simulations**. Each phase flips a "Coming soon" card on the track index to active.
- **Deferred decisions to confirm before Phase 2**:
  - Asset strategy (canvas-drawn solid colors, no external sprites).
  - Cross-link vs duplicate fundamentals content (default: link out with a one-paragraph recap).
  - Full TypeScript parity vs JS-only on the largest classes.
