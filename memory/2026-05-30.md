## Architecture notes (so future sessions don't relearn)

- **Tier-file convention** (from ARCHITECTURE.md): each `<tier>-demos.js` is
  self-contained — IIFE per demo, early-`return` if its canvas id is absent. A
  top-level `const`/`let`/`class` that shadows a `shared/utils.js` global
  (`lerp`/`clamp`/`map`/`Vector2D`) throws "already declared" at load and silently
  kills the whole file. Keep such helpers inside IIFEs.
- **Engine split:** truly cross-tier stuff (RNG, grid, renderer) lives in
  `roguelike/engine/`; each tier's *taught* algorithm (BSP, FOV, scheduler, ECS) is
  implemented in that tier's demos file and re-declared/assembled in the Simulations
  capstone.
- **Export opt-out:** roguelike demos are keyboard/canvas-driven; `export-demo.js` only
  renders button controls, so demos omit `data-demo-id` (no `bundles-*.js` for now).
  `rl_` namespace reserved for future bundles.

## Beginner tier — shipped (pt.2)

`roguelike/beginner.html` + `beginner-demos.js`, 5 demos: gridGlyphDemo, turnLoopDemo
(turn-based vs real-time side by side), bumpAttackDemo, combatLogDemo (turn order +
seeded log), and the capstone **oneRoomOneRatDemo** (sleeping rat wakes & hunts, bump
combat, win-by-stairs / lose-by-death, seed-reproducible).

Reusable `rl*` toolkit lives at the top of `beginner-demos.js` and will be re-declared
by later tiers (self-contained-tier convention): `rlKeyToStep`, `rlTryMove`
(move/attack/blocked), `rlStepToward`/`rlStepCandidates` (greedy pursuit, NOT
pathfinding), `rlInstallCanvasKeys` (click-to-focus + preventDefault), `rlLog`,
`rlFocusHint`, HP bars, flashes. All `rl*`-prefixed to avoid shadowing utils.js.

**Key gotcha learned:** keyboard demos need click-to-focus + `preventDefault` on
movement keys, or all demos grab arrows at once and the page scrolls. **Bug fixed:**
`turnLoopDemo.reset()` wasn't resetting the player position (only goblins), causing a
false "caught you!".

Verified in-browser (console clean): all 5 demos driven via synthetic keydowns; Demo 4
won deterministically (identical log across two same-seed runs); capstone played to a
win ("🏆 you descended"). Indexes + README updated; Intermediate card → "Building next".
